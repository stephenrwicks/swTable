swTable()

Here's my flexible and reactive data table component for vanilla html/css/js. No dependencies or framework - it's just a function.

Automatically displays json data in a table. Reactivity is built-in. When you update the data for a row, the <tr> will rerender.

It can also handle computed values. A column can be a combination of multiple keys.

Options and data can be set on initialization and can be modified after initialization.

Written with TS.
